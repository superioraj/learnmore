# 파이썬 3시간 끝내기
## 1차시 수업
* 출력문 & 주석
  ```
  print("Hello, World!")  # 출력문
  ```
* 입력문
  ```
  text=input('출력할 문장 입력:')
  print(text) 
  ```
* 변수와 데이터 타입
  ```
  name = "Alice"  # 문자열
  age = 25  # 정수
  height = 5.5  # 실수
  is_student = True  # 불리언
  ```
* 기본 연산
  - +, -, *, /, //. %, **
  ```
  age = 30
  sum = age + 5
  print(name + " will be " + str(sum) + " years old in 5 years.")
  ```
* if, elif, else 제어문
  ```
  number = 10
  if number % 2 == 0:
    print("The number is even.")
  elif number % 2 ==1:
    print("The number is odd.")
  else:
    print("다시 입력")
  ```
* For loop
  ```
  for i in range(5):
    print(i)
  ```
* While loop
  ```
  count = 0
  while count < 5:
    print(count)
    count += 1
  ```
  ```
* 자료구조: 리스트
  ```
  students = ["Alice", "Bob", "Charlie"]
  print(students[0])  # 리스트 요소 접근
  ```
* 자료구조: 튜플
  ```
  colors = ("red", "green", "blue")
  print(colors[1])
  ```
* 자료구조: 딕셔너리
  ```
  grades = {"Alice": 90, "Bob": 85, "Charlie": 92}
  print(grades["Bob"])

* 함수 정의 및 기본 호출
  ```
  def greet(name):
    return "Hello, " + name + "!"

  print(greet("Alice"))
  ```

* 3,6,9 게임 설계
  ```
  num = int(input("숫자를 입력하세요: "))

  for i in range(1, num + 1):
      number_str = str(i)
      contains_3 = '3' in number_str
      contains_6 = '6' in number_str
      contains_9 = '9' in number_str
  
      if contains_3 or contains_6 or contains_9:
          clap_count = number_str.count('3') + number_str.count('6') + number_str.count('9')
          print("짝" * clap_count)
      else:
          print(i)
  ```
